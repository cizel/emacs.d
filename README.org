** My Editor Workflow
*** Emoji 
    
    | 操作       | 快捷键    |
    | 插入 emoji | =SPC i e= |

*** Files
    
   | 操作             | 快捷键     |
   | 当前目录创建文件 | =<SPC> ff= |

*** Magit
    
   | 操作         | 快捷键  |
   | 查看文件变动 | =<TAB>= |
   | 查看提交日志 | =ll=    |
   | commit 操作  | =cc=    |
   | 切换分支     | =bb=    |
